## Installation

1. Clone this repo to your local machine

```sh
    git clone https://github.com/Gonksy/na-demo
```

2. Navigate to the project directory

```sh
    cd na-demo
```

3. Install dependencies

```sh
    npm install
```

4. Run the start command

```sh
    npm run start
```
